# Práctica 4 - Android: Navegación y paso de datos entre actividades

Aplicación Android nativa (Kotlin) que captura datos de un formulario y muestra un resumen para confirmar antes de regresar a la pantalla principal.

## Funcionalidad

- Formulario con campos: nombre, edad, ciudad y correo electrónico.
- Navegación desde `MainActivity` a `ResumeActivity` usando `Intent`.
- Visualización de datos en pantalla de resumen.
- Botón de confirmar que regresa `RESULT_OK` a la actividad principal.
- Botón para volver a editar sin confirmar.
